# Reference samples — DO NOT use as demo output

These are **frozen sample artifacts** captured from a prior run against
`juice-shop/juice-shop`, kept for reference and regression-testing. They are
not produced by the live demo and not consumed by it.

When you run `make learn` (or `python guided_demo.py`), fresh files are
written to `prototype/runs/<repo-slug>/<timestamp>/` — that's the only place
the demo reads or writes during the hackathon.
